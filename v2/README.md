## 1. 角丸（Radius）の設定

`border-radius` は直接使用せず、`--radius` を使用することを推奨します。

| 項目 | 指定方法 | 例 |
| --- | --- | --- |
| **推奨** ✅ | `--radius` を使用 | `div { --radius: 10px; }` |
| **非推奨** ❌ | `border-radius` を使用 | `div { border-radius: 10px; }` |

---

## 2. スムースコーナー（Superellipse）

角の曲線をより滑らかにする「スムースコーナー」の設定です。

* **有効にする（推奨）**
デフォルトで適用されます。数値を調整する場合は以下を記述します。
```css
--elm2-corner-superellipse: 1.6;

```


* **無効にする**
通常の角丸にしたい場合は `1` を指定してください。
```css
--elm2-corner-superellipse: 1;

```



> [!NOTE]
> ブラウザがスムースコーナーに非対応の場合、自動的に無効（値：1）として処理されます。

## 3. 開発用ファイル（test.css）の扱い

`test.css` は、開発中に背景を見やすくするための補助ファイルです。

### 仕様

* 他のCSSで `background` や `background-color` が指定されている場合、**そちらの指定が優先されます。**

### 実装完了後の作業

本番公開前、または実装完了時に必ず以下の**いずれか**を実施してください。

1. HTML等での `test.css` の**読み込み（インポート）を削除**する。
2. `test.css` 内の記述を**すべてコメントアウト**する。
